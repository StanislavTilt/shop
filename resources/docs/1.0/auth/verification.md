<a name="verification"></a>
## Верификация

### `POST` **Конечная точка**

```text
auth/verification/verify
```

### Тело запроса


|Имя поля|Валидация|Описание|
|:-|:-|:-|
|phone|`required|string`|Номер телефона|
|code|`required|int|digits:5`|Код подтверждения из смс|

---

### Ответ

> {success} Успешный ответ. Код `200`

```json
{
    "data": {
        "id": 42,
        "name": "Sheroz",
        "avatar": "http://my-italy.loc/",
        "nickname": null,
        "phone": "+992929676791",
        "email": null,
        "password": "$2y$10$OOcrfqeQwhvFjNA1CkzVoewRmXHNJznbPVm\/PNXE9Z1blRcItDE0W",
        "status": 2
    },
    "token": "20|LY8C6wmM29BP42hCFRh3ynncK6kvK4wZU9BNOjGK"
}
```

> {danger} Ответ с ошибкой. Код `403`

Причина `Код потверждения уже отправлялся ранее`

```json
{
    "message": "Код подтверждения уже отправлен.",
    "errors": {
        "error": [
            "Код подтверждения уже отправлен."
        ]
    }
}
```


<a name="resend-code"></a>
## Повторная отправка кода подтверждения

> {warning} Запросить код можно по истечению **2** минут после последней попытки. А после **пятой** попытки пользователю выдается бан на день.   

### `POST` **Конечная точка**


|Метод|URI|
|:-|:-|:-|
|POST|`auth/verification/resend-code`

### Body


|Имя поля|Валидация|Описание|
|:-|:-|:-|
|phone|`required|string`|Номер телефона|

---

### Ответ

> {success} Успешный ответ. Код `200`

```json
{
    "message": "Код подтверждения отправлен на номер телефона"
}
```

> {danger} Ответ с ошибкой. Код `403`

Причина `Код потверждения уже отправлялся ранее`

```json
{
    "message": "Код подтверждения уже отправлен.",
    "errors": {
        "error": [
            "Код подтверждения уже отправлен."
        ]
    }
}
```

<a name="get-code"></a>
## Получение кода верификации

> {info} После успешной регистрации пользователя, нужно верифицировать его. О том как это сделать, можете узнать в разделе [Верификация](verification)

### `POST` **Конечная точка**
```text
/auth/verification/get-code
```

### Тело запроса

> {warning} Этот метод исключительно для тестирования, и в релизной версии будет удалён.

|Имя поля|Валидация|Описание|
|:-|:-|:-|
|phone|`required|string`|Номер телефона

### Ответ

> {success} Успешный ответ. Код `200` 

```json
{
    "code": 21347
}
```
